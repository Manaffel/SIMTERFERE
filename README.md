SIMTERFERE
==========

SIMTERFERE is a is a Python-based, data-driven interferometric simulator designed to reproduce VLTI GRAVITY on-axis observations using their associated ancillary data.

Installation
------------

- pip install simterfere

or
- git clone https://github.com/Manaffel/SIMTERFERE.git

### Requirements
- astropy
- corner
- matplotlib
- numpy
- poppy
- scipy
- telfit
- torch

Tutorial
--------
In examples you can find "tutorial.py". It is a small script explaining the basic setup of SIMTERFERE.

You need to download two files and add them to "data/tutorial/".
